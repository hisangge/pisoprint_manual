üñ®Ô∏è How to Print Your Document

Method 1: Print from USB Drive üíæ

1 Prepare Your File
  • Save your document as a PDF file on a USB flash drive
  • Make sure the file is under 50MB
2 Insert USB Drive
  • Locate the USB port on the kiosk (usually on the front or side)
  • Insert your USB drive - it will be detected automatically (1-2 seconds)
3 Select Your File
  • The touchscreen will display all PDF files on your USB drive
  • Tap the file you want to print
  • The system will process the file and show the page count

Method 2: Upload via WiFi üì±

1 Connect to WiFi
  • On your phone or laptop, go to WiFi settings
  • Find and connect to: "PisoPrint_Kiosk"
  • Enter password: PisoPrint2025
2 Open Web Browser
  • Open any web browser (Chrome, Safari, Firefox, Edge)
  • Go to: http://pisoprint.local/mobile/upload
3 Upload Your File
  • Tap "Choose File" and select your PDF document
  • Tap "Upload" and wait for completion
  • You'll see a confirmation with page count and estimated cost
4 Go to Kiosk
  • Disconnect from WiFi (optional)
  • Go to the kiosk touchscreen
  • Select your uploaded file from "Recent Uploads"
üîí Security Note: For security reasons, external devices can only access the upload page. The full kiosk interface is restricted to the kiosk touchscreen only.

üîß Troubleshooting

‚ùå Problem: Coin not accepted

‚úÖ Solution: Use clean, undamaged Philippine peso coins. Try a different coin. Make sure it's one of the accepted denominations (‚Ç±1, ‚Ç±5, ‚Ç±10, ‚Ç±20).

‚ùå Problem: File upload failed

‚úÖ Solution: Make sure your file is in PDF format and under 50MB. Check your WiFi connection or try re-inserting your USB drive.

‚ùå Problem: "File too large" error

‚úÖ Solution: Your file exceeds 50MB. Try compressing the PDF or splitting it into smaller files.

‚ùå Problem: Balance not updating after inserting coin

‚úÖ Solution: Wait 5-10 seconds. The touchscreen should update automatically. If not, notify staff.

‚ùå Problem: Print job stuck or not printing

‚úÖ Solution: Wait 30 seconds for system recovery. If still stuck, tap "Cancel" and notify staff. Paper jam or printer error may have occurred.

‚ùå Problem: Can't connect to WiFi

‚úÖ Solution: Verify you're connecting to "PisoPrint_Kiosk" with password "PisoPrint2025". Try turning WiFi off and on again on your device.

‚ùå Problem: System frozen or unresponsive

‚úÖ Solution: Wait 30 seconds for auto-reset. If screen remains frozen, notify staff or administrator immediately.
